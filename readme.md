# @lucy/lucy-repl

a REPL for Lucy source files.

> Note that this is in very early development and doesn't work yet.

## Usage

```html
<lucy-repl tab-size="2">
  <pre slot="lucy"><code>state enabled {
  toggle => disabled
}

initial state disabled {
  toggle => enabled
}</code></pre>
</lucy-repl>
```
